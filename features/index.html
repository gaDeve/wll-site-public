Points & Rewards Engine

A configurable points system with expiry management and order-based earning.

  • Earn rate and redeem rate configurable per program
  • Points expiry with bucket-based tracking (FIFO consumption)
  • Min order thresholds for earning and redemption
  • Max redeem percentage and discount caps
  • Channel-specific rules (POS, online, mobile)
  • Idempotent transactions with hash-based deduplication

Vouchers & Promotions

  • Lifecycle: Inactive → Active → Paused → Archived
  • Auto stock management with code generation
  • Expiry tracking at offer and code level

SMS Campaigns (BYOP)

Bring your own SMS provider and target customer segments with personalized messages. Available on all plans.

  • Connect any SMS gateway (Twilio, MessageBird, or custom)
  • Audience targeting by segment, status, or custom criteria
  • Template-based message personalization with variables
  • Flexible auth: API key, Bearer token, or Basic auth
  • Delivery status tracking (Pending, Sent, Failed)
  • Campaign voucher distribution with SMS notification

Technical Foundation

Enterprise-grade architecture

Built on Spring Boot, PostgreSQL, and Keycloak with multi-tenant isolation.

Multi-Tenant Architecture

Run multiple brands from a single platform. Each tenant gets isolated data, custom subdomain, and separate admin access.

API-First Design

Full REST API access on all plans. Integrate with your POS, e-commerce, or mobile app from day one.

Enterprise Security

Role-based access control, secure authentication, and data encryption. Your customer data is always protected.

Subscription Management

Flexible billing with plan tiers, prorated upgrades/downgrades, auto-renewal, and invoice management.

Automated Operations

Points expiry, SMS dispatch, segment upgrades, and cashback evaluation run automatically on schedule.

Audit Trail

Complete audit logging for points operations, card changes, and admin actions for compliance and transparency.

API Reference

RESTful API on all plans

Full API access included on every plan. Integrate your POS, e-commerce, or mobile app from day one.

API Endpoints
POST/api/v1/cards/{id}/earnAdd points to a card
POST/api/v1/cards/{id}/redeemRedeem points from a card
GET/api/v1/customersList customers with pagination
POST/api/v1/customersRegister new customer with card
GET/api/v1/vouchers/{code}/redeemRedeem voucher code
GET/admin/v1/analytics/summaryGet dashboard analytics
POST/admin/v1/campaigns/smsSend SMS campaign
GET/admin/v1/points/expiringList expiring points
